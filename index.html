Posted on 2021-02-10

有一個地下市場的吐鈔機販賣部，早上會收購吐鈔機，晚上會販賣吐鈔機。

現在有 $N$ 種吐鈔機，第 $i$ 種會在第 $D_i$ 天以 $P_i$ 元販賣，每天會吐出 $G_i$ 元的鈔票，並且可以用 $R_i$ 元的二手價賣出，而每個人最多只能擁有一台吐鈔機。

現在你手中有 $C$ 元，並且你打算在市場交易 $D$ 天，請問在第 $D+1$ 天你最多可以賺多少錢 ?

考慮 dp 的話，可以推出一個 $O(N^2)$ 的做法，轉移式如下 :
$$ dp_i = \min_{j=1}^{j< i, dp_j\geq P_j}(dp_j+G_j*(D_i-D_j-1)+R_j-P_j) $$
其中 $dp_i$ 為在買進第 $i$ 台吐鈔機那天的早上最多可以賺的錢。

把式子再稍微統整一下會變成 :
$$ dp_i = \min_{j=1}^{j< i, dp_j\geq P_j}(G_jD_i-G_j*(D_j+1)+R_j-P_j+dp_j)$$
這時候可以將轉移式看成一條直線 $mx+c$，其中 $m$ 為 $G_j$，$x$ 為 $D_i$，$c$ 為 $G_j*(-D_j-1)+dp_j+R_j-P_j$。

就變成了經典的斜率優化問題，而複雜度從原本的 $O(N^2)$ 變成 $O(NlogN)$。
而我是用李超線段樹來維護某個點在直線集合中代入的極值。

有$N$個俄羅斯娃娃, 每個俄羅斯娃娃的寬度為$w_i$, 高度為$h_i$。

如果有一個娃娃$x$的寬度及高度皆嚴格小於另一個娃娃$y$ ($w_x<w_y$, $h_x<h_y$),
就可以把娃娃$x$放進娃娃$y$裡面。
問最多可以疊出幾層娃娃。

裸二維LIS。
做法就是先對其中一維由小到大排序, 再對另一維做LIS。

有個要注意的點就是 :
兩個其中一維相同的娃娃不能被放在同一個遞增序列裡, 而避免的方式就是第二個維度由大到小排序。

Tioj1082

Posted on 2020-12-13

有$N$堆石頭，每堆石頭有 $B_i$ 個黑色石頭以及 $W_i$ 個白色石頭。
每次操作可以把其中一堆的一個石頭搬到其他堆。
求使每堆石頭都只有一種顏色的最少操作次數，如果無法達成則輸出$-1$。

$N \leq 10^6$, $B_i +W_i \leq 2^{31}$

最直覺的方法就是把每堆數量較少的顏色搬到其他堆去。
所以答案就是$\sum {\mit min}(B_i, W_i)$。

有兩個比較特別Case要判 :

  • 每堆數量較少的顏色都一樣 ( 假設都是黑色比較少 ) :
    這時候要挑其中一堆石頭 $x$ 放搬走的黑色石頭，並且把那堆的白色石頭搬走，
    而這時候花費會變成$\sum B_i+(W_x-B_x)$。
    可以觀察到前面的$\sum B_i$不變，所以要找的就是${\mit min}(W_x-B_x)$。
  • 只有一堆石頭，且有兩種顏色的石頭 :
    這時候不存在一種分法使每堆石頭只有一種顏色。
